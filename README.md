# Overview

Todo app using Next.js and Nest.js.

# frontend

https://localhost:3000

## Installation

```bash
npm install
```

## Run

```bash
npm run dev
```

# backend

https://localhost:3001

## Create certificates

### mkcert

```bash
brew install mkcert
mkcert -install
```

### Create certificate for localhost

```bash
cd backend
mkcert localhost
```

## Installation

```bash
npm install
```

## Run

```bash
npm run start:dev
```
